# Organisasjoner

Organisasjoner er et felles register på tvers av alle løp i arrangementet.

## Opprette organisasjoner

Ved import av påmeldinger så importeres også organisasjoner, men ofte med mangelfull kontaktinformasjon.
Kjør oppgaven [Importer organisasjoner](/nb/tasks/import-classes) for å hente mer kontaktinformasjon.



## Felter

| Felt                 | Beskrivelse                                                                                 |
|----------------------|---------------------------------------------------------------------------------------------|
| Navn                 | Organisasjonens navn                                                                        |
| Organisasjonstype    | Organisasjonens type                                                                        |
| Foreldreorganisasjon | Angi foreldreorganisasjon for å lage et organisasjonshierarki. Benyttes foreløpig ikke til noe. |
| Land                 | Organisasjonens land                                                                        |
| E-post og telefon    | Brukes i forbindelse med fakturering                                    |
| Adresse              | Brukes i forbindelse med fakturering                                                        |
| Identifikatortype/Identifikator | [Identifikatorer](/nb/resources/common/identifiers)
