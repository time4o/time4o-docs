# Løp

Et arrangement kan bestå av flere løp. I flerdagersløp er det typisk ett løp pr dag, 
men det er også mulig å opprette flere løp for samme dag, f.eks. om det er prolog og jaktstart samme dag,
eller hvis man arrangere både stafett og individuelt.


## Felter

| Felt         | Beskrivelse                                         |
|--------------|-----------------------------------------------------|
| Navn         | Løpets navn.                                        |
| Løpsnummer   | Gjelder kun flerdagersløp.                          |
| Identifikatortype/Identifikator | [Identifikatorer](/nb/resources/common/identifiers) |
