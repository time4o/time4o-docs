# {{ $frontmatter.title }}

### Beskrivelse

Genererer resultatliste for [poengløp](/nb/guide/tutorials/point-race) i excel-format.

Excel-filen viser poeng pr. post og løype avhengig av type poengberegning, samt totale poeng. Listen er sortert eller usortert basert på innstillingen på klassen.

Kolonneoverskriften for poengposter viser postkoden som standard. Dette kan overstyres av feltet "Beskrivelse" på løypepostene.

### Konfigurasjon

| Innstilling      | Beskrivelse                                                                  |
|------------------|------------------------------------------------------------------------------|
| Løp / Klasse     | Klasser som resultatlisten skal vise. Det genereres én excel-fil pr. klasse. |
| Vis organisasjon | Vis kolonne med påmeldingens organisasjon.                                   |
