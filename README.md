# Time4o Dokumentasjon

Dette er kildekoden til [Time4o sin Dokumentasjon](https://docs.time4o.com).
Kildekoden består i hovedsak av en katalogstruktur og Markdown-filer 
og bruker [VitePress](https://vuejs.github.io/vitepress/v1/) til å generere statiske sider.

Føl deg fri til å bidra med ny dokumentasjon, forbedre eksisterende dokumentasjon eller rette skrivefeil som det er mange av.
